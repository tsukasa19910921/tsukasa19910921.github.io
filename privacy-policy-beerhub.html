BeerHub プライバシーポリシー

最終更新日: 2025年11月22日

1. はじめに

BeerHub（以下「当アプリ」）は、ユーザーの皆様のプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、当アプリがどのような情報を収集し、どのように使用するかを説明するものです。

2. 収集する情報

当アプリは、以下の情報を収集します：

  • アカウント情報（メールアドレス、表示名）
  • プロフィール情報（アイコン画像、SNSアカウント、コメント）
  • 位置情報（チェックイン時のGPS座標）
  • カメラ画像（QRコードスキャン用）
  • 写真ライブラリ（プロフィール画像および店舗アイコン設定用）
  • 利用状況（チェックイン履歴、評価情報）
  • デバイス情報（Push通知トークン、デバイスの種類、OSバージョン）

3. 情報の利用目的

収集した情報は、以下の目的で使用します：

  • サービスの提供と改善
  • チェックイン機能、店舗検索、ランキング表示などの提供
  • ユーザー体験のパーソナライズ
  • ユーザーサポート
  • 不正利用の防止
  • Push通知の送信（チェックイン完了、ランクアップ時など）
  • 統計データの作成（個人を特定できない形式）
  • アプリのバグ修正とパフォーマンス最適化

4. 第三者サービスへのデータ送信

当アプリは、以下の第三者サービスを使用しており、これらのサービスにデータが送信される場合があります：

4.1 Firebase (Google)

  • サービス: Firebase Authentication, Firestore Database, Firebase Storage, Firebase Cloud Functions
  • データタイプ: ユーザーアカウント情報、チェックイン履歴、画像データ
  • 目的: ユーザー認証、データの保存・管理、バックエンド機能の提供
  • プライバシーポリシー: https://firebase.google.com/support/privacy

4.2 Expo SDK

  • サービス: Expo プッシュ通知、アプリ更新管理
  • データタイプ: デバイストークン、アプリバージョン情報
  • 目的: プッシュ通知の配信、アプリの更新管理
  • プライバシーポリシー: https://expo.dev/privacy

5. 情報の共有

当アプリは、以下の場合を除き、第三者に個人情報を提供しません：

  • ユーザーの同意がある場合
  • 法令に基づく場合
  • サービス提供に必要な範囲で業務委託先（Firebase等）に提供する場合

6. 位置情報について

チェックイン機能を使用する際、ユーザーの位置情報を収集します。位置情報は、店舗への実際の訪問を確認する目的、および近くの店舗を検索・表示する目的でのみ使用され、ユーザーの同意なしに他の目的で使用することはありません。

位置情報の使用は、ユーザーが許可した場合のみ行われます。iOSの設定からいつでも位置情報の使用を無効にすることができます。

7. カメラとフォトライブラリへのアクセス

本アプリは、以下の目的でカメラとフォトライブラリにアクセスします：

  • カメラ: QRコードスキャンによるチェックイン
  • フォトライブラリ: プロフィール画像および店舗アイコンの設定

これらのアクセスは、ユーザーが許可した場合のみ行われます。iOSの設定からいつでも無効にすることができます。

8. プッシュ通知

本アプリは、以下のタイミングでプッシュ通知を送信する場合があります：

  • チェックイン完了時
  • ランクアップ時
  • 重要なお知らせ

プッシュ通知は、ユーザーが許可した場合のみ送信されます。iOSの設定からいつでも無効にすることができます。

9. データの保管

収集した個人情報は、Firebase（Google Cloud Platform）上に安全に保管され、適切なセキュリティ対策を講じています。

  • 保存場所: ユーザーデータは Firebase (Google Cloud) に安全に保存されます
  • 暗号化: データは転送時および保存時に暗号化されます
  • アクセス制限: ユーザーデータへのアクセスは、認証されたユーザーのみに制限されています
  • 保持期間: ユーザーがアカウントを削除するまで、データは保持されます

10. ユーザーの権利

ユーザーは、以下の権利を有します：

  • 自身の個人情報へのアクセス
  • 個人情報の訂正・削除
  • アカウントの削除
  • データのエクスポート（データポータビリティ）

11. Cookie等の技術

当アプリは、サービスの改善のためにCookieやローカルストレージなどの技術を使用する場合があります。これらの技術は、ユーザーの設定情報やアプリの状態を保存するために使用されます。

12. 未成年者について

13歳未満の方は、保護者の同意なく当アプリを使用することはできません。13歳未満の子どもから故意に個人情報を収集することはありません。13歳未満の子どもの個人情報を収集したことが判明した場合、速やかに削除いたします。

13. プライバシーポリシーの変更

当アプリは、必要に応じて本プライバシーポリシーを変更することがあります。重要な変更がある場合は、アプリ内またはメールで通知いたします。変更後のプライバシーポリシーは、本ページおよびアプリ内で公開されます。

14. お問い合わせ

プライバシーポリシーに関するご質問、個人情報の削除リクエスト、その他のお問い合わせは、以下までご連絡ください：

  • GitHub Issues: https://github.com/tsukasa19910921/tsukasa19910921.github.io/issues
  • メール: お問い合わせは上記GitHub Issuesからお願いいたします

個人情報の削除リクエストについて: アカウント削除をご希望の場合は、アプリ内の「設定」→「アカウント設定」から削除が可能です。それ以外の個人情報削除に関するご要望は、上記GitHub Issuesにてご連絡ください。

15. 準拠法

本プライバシーポリシーは、日本国の法律に準拠し、解釈されるものとします。
